Welcome to Cynthia

Search for deepest bonds in the living world

Cynthia allows to biologist and biotechnologist researcher to query biological databases with a simple and dynamic user-interface
After specifying all the requests the results are graphically shown as a graph
All the connection between searched elements will be drawn in a simple and clear view

How Cynthia can help your researches

Query Engine

A built-in mechanism allows to build queries for the biological database and get results, in a completely transparent way for the user.

User Friendly

Specific knowledges about databases and query language are not required. The user has to simply select from a common and simple web form what he is searching for.

Dynamic

The researches can be taken dynamically and the user is guided in the process of building the query. Many different searches can be performed on different elements, with different targets.

Cloud-based

Cynthia runs on a server just like a classic website. It is just necessary to have a web browser (just like the one you are using now!) and the URL of the application.

Frequently Asked Questions


Do I need informatic skills or knowledges?

Absolutely not! The informatic skills of the enGeneus developing team allowed us to build a tool that can be used by biologist and biotechnologist researchers letting them forget code and query languages!

Can I save my search results and see them later?

As former president Obama would say, yes, you can! You can export the cypher query, the raw query data in cynthia format, the whole offline results and the CSV data. You can only directly import the cypher query, the raw query data or the offline results
